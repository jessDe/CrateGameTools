# Crate Game Tools
A small tool for the VRChat Crate Game

## Features
- Save Management : Captures Backup Saves from Clipboard or manually pasting
- Maintenance Detector : Warns about upcoming maintenance so you have time to save
- Timer Alarm : reminding to copy the backup save

## Planned Features
- Getting saves from VRChat Log file (requires game update)
- Vending Machine and Boosters reminder
- Custom Save Location + Cloud Backup
